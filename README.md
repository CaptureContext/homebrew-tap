# capturecontext tap

Homebrew tap for CaptureContext tools

## [fxcodex](https://github.com/capturecontext/fxcodex)

A macOS CLI for managing isolated Codex workspaces.

### Requirements

- macOS 14 or later

### ⌘ Install

```bash
brew install capturecontext/tap/fxcodex
```

### ⌘ Update

```bash
brew upgrade fxcodex
```
